# UUID JS

Simple utility for generating collision resistant UUID within the web browser

### Installing with NPM

```
npm install uuid-js
```

### Installing with Yarn

```
yarn add uuid-js
```

### Example usage

```javascript
import uuid from "uuid-js";

const Uuid = uuid();
```

### Run unit tests

```
yarn test
```
